# 本地化战略人才分析 · 算法与架构

围绕从「本地新闻分发」到「本地 AI 信息与服务平台」的演进路径，盘点所需的算法与架构人才（已脱敏，所有公司 / 产品代号均使用通用化表述）。

## 核心叙事

```
本地内容平台  →  本地 AI 信息平台  →  本地 AI 服务平台
   Consume         Discover · Decide          Act
```

从 Local News 升级为 **Local × AI**：横向构建信息平台，纵向打通服务平台。
